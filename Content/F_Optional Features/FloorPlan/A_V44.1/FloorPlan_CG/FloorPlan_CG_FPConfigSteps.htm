Floor Plan Configuration Steps

  1. Within EMS, upload your floor plan indicators and floor plans as outlined in the EMS Desktop Setup Guide: Core Configuration-Facilities section.

Note:  Optimal floor plan image resolution is 1024x768.  Valid file types are .png, .jpg, .jpeg, .jpe, or .jfif.

  1. Using the Floor Plan Configuration Utility, map indicators to room locations on your floor plans. 
  2. To enable the Floor Plan Module for use with EMS Web App, go to System Administration > Settings > Parameters > Everyday User Applications and locate the parameter – ‘Floor Mapping – Floor Mapping Web Service URL’.  Enter the URL of your Floor Plan Web Service (step 10 in the Installing the Floor Plan Web Service section).
  3. The View Floor Map Web Security Template Role must be granted (within Configuration > Web > Web Security Template) in order for web users to see an interactive floor map on EMS Web App.
  4. To configure the Floor Plan Module for use with the EMS Kiosk (optional module), go to System Administration > Settings > Parameters > Everyday User Applications and locate the parameter – ‘Floor Plans – Floor Plan Web Service URL’. Enter the URL of your Floor Plan Web Service (step 10 in the Installing the Floor Plan Web Service section.)
