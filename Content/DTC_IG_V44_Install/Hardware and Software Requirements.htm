Database Server 

Supported Platforms

Windows 2012 Server
Windows 2012 Server R2
Windows 2008 Server R2

Minimum System Requirements

Processor: 2.0GHz and 4 cores or faster
Memory: 8GB or more*
Hard-Disk Space: 2 separate physical drives of at least 6GB; one for data, and the other for transaction logs (SCSI drives w/9ms access or better and RAID strategy for fault tolerance recommended)
Software: Microsoft SQL Server 2014, Microsoft SQL Server 2012, Microsoft SQL Server 2008 R2
Network Card: 100 / 1000Mbps ethernet, switched

 

*Specs apply for up to 100 concurrent users. Increased specs are required for 100+ concurrent users.
