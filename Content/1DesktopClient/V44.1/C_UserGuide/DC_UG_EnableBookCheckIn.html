Enabling Booking Check-in

Tip: For an explanation of this feature, see Understanding Booking Level Check In.
To manage space that uses booking check in, see Managing Booking Level Check-In

 

  1. On the EMS menu bar, click Configuration > Facilities > Rooms.
  2. On the Building dropdown list, select one of the following to configure a room — (all) Buildings, a specific Building, a specific area, or a specific view.
  3. Select and edit the room for which booking level check in needs to be enabled.
  4. Check the box for Requires Check In. The minutes prior to booking event start is the amount of time before a booking in the specified room will be open for check in.

Note: This type of check-in can be done to multiple rooms at the same time using the Wizard on the Rooms screen.
Anyone with access to the desktop client can check into a booking earlier than the time assigned to the room.
