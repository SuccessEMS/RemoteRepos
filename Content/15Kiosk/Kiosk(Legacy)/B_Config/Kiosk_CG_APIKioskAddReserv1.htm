API Functions - Kiosk Add Reservation 1 and Reservation 2

KioskAddReservation

Description

Adds a new event for a given set of parameters.

Request Parameters

Name

Description

Type

String

String

Integer

Group ID

Integer

EventTypeID *

Event Type ID

Integer

DateTimeList *

XML Date Time List

String

Response Elements

Name

Description

Type

ReservationID

Reservation ID that was created in EMS

Integer

Notes

* Indicates required field.

DateTimeList example: <Dates><Date requestedDate="2010-10-27" startTime="10:42:00" endTime="20:00:00" /></Dates>.

For a reservation spanning multiple days just pass in additional <Date /> nodes in the DateList XML.

Returns a -1 if the Reservation was NOT created.
