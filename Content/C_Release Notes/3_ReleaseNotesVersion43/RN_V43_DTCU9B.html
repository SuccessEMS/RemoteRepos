EMS DTC Release Notes V43 Update 9b

Area Description
Web Client, Departments
Unable to Edit or Delete Departments on Web Client
When administrative users selected Configuration > Administration > Departments and clicked Delete on a department in the list, the department was not deleted and an error occurred, "Invalid column name 'Confirmations." When they attempted to edit a department, their changes were not saved and the Edit window did not close.

Fix: Modified validations to ensure that deleting and editing departments is successful. 

27835, 24978 FIX ONLY IN v43
Web Client, Group Notification Rules
Header Message and Time Zone Not Saved for Group Notification Rules on Web Client
When administrative users selected Configuration > Administration > Group Notification Rules and created or edited a rule, their selections for Header Message and Time Zone were not saved.

Fix: Modified validations to ensure that header message and time zone selections for Group Notification Rules are saved. 

16314 FIX ONLY IN v43
Web Client, Transaction Report
Voided Transactions Persist on Transaction Report on Web Client
When user ran a Transaction Report from the Web Client, the report showed voided transactions, unlike in the {{product_dtc}}.

Fix: Modified the Transaction Report to ensure that it does not include voided transactions. 

15199 FIX ONLY IN v43
