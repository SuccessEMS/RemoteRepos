Installing .NET Framework 3.5 and .NET Framework 4.5

  1. In Server Manager, click Manage and then select Add Roles and Features to start the Add Roles and Features Wizard.

  1. In the Confirm Installation Selections interface, review the selections and then click Install.
  2. Allow the installation process to complete and then click Close.
  3. On the Select installation type screen, select Role-based or feature-based installation.
  4. Select the target server.
  5. On the Select Features screen, check the box next to .Net Framework 3.5 Features.

​​​​​​​

  1. In the Select Features interface, expand .NET Framework 3.5 Features.
  2. Once expanded, there will be three check boxes. One for .NET Framework 3.5 and the other two for HTTP Activation and Non-HTTP Activation. Check the box next to .NET Framework 3.5 and click Next.

Installing .NET Framework 4.5: Regics Only

  1. In Server Manager, click Manage and then select Add Roles and Features to start the Add Roles and Features Wizard.

  1. On the Select installation type screen, select Role-based or feature-based installation.
  2. Select the target server.
  3. On the Select Features screen, check the box next to .Net Framework 4.5 Features.
  4. In the Select Features interface, expand .NET Framework 4.5 Features.
  5. Once expanded, there will be three check boxes. These will be .NET Framework 4.5, APS.NET 4.5 and WCF Services. Check the boxes next to .NET Framework 4.5 and ASP.NET 4.5 then click Next.

​​​​​​​

  1. In the Confirm Installation Selections interface, review the selections and then click Install.
  2. Allow the installation process to complete and then click Close.
