What is an Everyday User Reservation?

Concept: An Everyday User Reservation is any reservation that a user has submitted through {{product_webapp}}.

The everyday user reservation searching tool in the {{product}} Desktop Client is identical to the Browser searching tool with one exception: when the Browser opens for the first time, the Browse for function is set to Reservations whereas the Everyday User Reservation tool is set to Everyday User Reservations. As a result, although this section is written from the perspective of using the Browser, all the information can be applied when using the Everyday User Reservation searching tool.

To open this tool in {{product}} Desktop Client, click the Everyday User Reservations icon on the {{product}} toolbar.
