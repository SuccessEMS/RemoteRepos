Installing or Upgrading the EMS Campus Planning Interface

Page summary display

template('MindTouch/Controls/PageOverview');

Important:  Before beginning the installation process, please install or upgrade your EMS databases as outlined in the EMS Installation Instructions.

  1. Manually uninstallany previous versions of the EMS Campus Web Client/EMS Campus Planning Interface on your web server. 
  2. Verify that the prerequisite software has been installed.
  3. Download the EMSCampusPlanningInterface.msi file onto the web server that will be running EMS Campus Planning Interface.
  4. Run EMSCampusPlanningInterface.msi.
  5. The first screen welcomes you to the Campus Planning Interface Setup Wizard.  Click Next to begin the installation process.  The Destination Folder screen will appear.

  1. Select the destination folder.  The installation process will create a new physical directory on your web server based on the destination folder path entered (“Campus Planning Interface” in the example above.)  Click Next.

Note:  EMS Campus Planning Interface should not be installed in the same physical directory as other EMS web-based products OR under a site running another version of EMS Campus Planning Interface.

  1. The SQL Server and database information screen will appear.

 

 

  1. Enter your SQL Instance (i.e. Server) Name.
  2. Enter your Database Name (typically named “EMS”). Click Next.
  3. The Virtual Directory information screen will appear. 

  1. The Virtual Directory Name will default to the destination folder specified in Step 6.  It is recommended that you keep the default setting.  The installation process will create a virtual directory on your web server based on the virtual directory entered (“Campus Planning Interface” in the example above.)  Click Next.

Note:  EMS Campus Planning Interface should not be installed in the same virtual directory as other EMS web-based products OR under a site running another version of EMS Campus Planning Interface.

  1. The Ready to install Campus Planning Interface screen will appear.  Click   to install EMS Campus Planning Interface.
  2. The Completed the Campus Planning Interface Setup Wizard screen will appear.  Click Finish.
