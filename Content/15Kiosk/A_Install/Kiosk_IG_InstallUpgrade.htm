Installing/Upgrading the EMS Kiosk App

To install the EMS Kiosk App:

  1. Verify that the System Requirements and Prerequisites for EMS Kiosk App have been fulfilled.
  2. Verify that EMS Platform Services has been installed.
  3. Load the EMS.Kiosk.App.msi file onto the web server that will be running the EMS Kiosk App.
  4. Run EMS.Kiosk.App.msi.
  5. The first screen welcomes you to the EMS Kiosk Setup Wizard. Click the Next button to begin the installation process.

Welcome Screen to the EMS Kiosk Setup Wizard

  1. The Destination Folder screen will appear. 

Note: The default install folder for the EMS Kiosk App is not under wwwroot but under C:\Program Files. However, you can change the location of the default installation folder.

Destination Folder

Note: EMS Kiosk App should not be installed in the same physical directory as other EMS web-based products OR under a site running another version of the EMS Kiosk App.

  1. In the EMS Application Settings screen, enter the EMS Platform Services URL and specify the Virtual Directory Name.

EMS Application Settings Screen

  1. Enter your EMS Kiosk App path and click Next. The Virtual Directory screen will appear.

Virtual Directory

  1.  Enter the name you want to give to the virtual directory. Click Next. The Ready to Install screen will appear.

Ready to Install Screen

  1. Click the Install button to install EMS Kiosk App.
  2. Reset the client secret in the EMS Platform Admin UI:
  1. Visit the Admin UI for you EMS Platform Services install (e.g., https://my-company.com/EmsPlatform/admin).
  2. Select Integrations in the left navigation.
  3. Select EMS Kiosk from the integrations list.
  4. Click the Reset Secret button.

Reset Secret Button

  1. A new Client Secret is generated following the reset.

New Client Secret

  1. Copy the generated Client Secret after successful reset.

Note: You will not be able to retrieve this secret later, instead you will have to reset the secret again to generate a new secret.

  1. Copy the client secret into the EMS Kiosk install:
  1. Find appsettings.json in the EMS Kiosk installation folder. 
  2. Find the ClientSecret entry, which is blank ("") by default. 
  3. Paste the secret into ClientSecret value (leave the comma at the end of the line).

Sample ClientSecret Value

  1. Save and close the file.
  1. Open the EMS Kiosk in a browser and follow the instructions in Staring the EMS Kiosk App.
