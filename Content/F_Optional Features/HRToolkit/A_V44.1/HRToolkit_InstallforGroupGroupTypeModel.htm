Installing HR Toolkit for the Group/Group Type Model

Once you have obtained the installation files, perform the following steps on the MS SQL Server that hosts your EMS database Using Microsoft SQL Server Management Studio.

  1. Execute the HRTK Staging DB Install G_GT_WU.sql script against your master database.  This will create the EMS_Staging database containing one table named tblPeople.
  2. Execute the HRTK Update Script G_GT_WU.sql script against your EMS database.  This will create a stored procedure in your EMS database named HRTK_Update_Group.
