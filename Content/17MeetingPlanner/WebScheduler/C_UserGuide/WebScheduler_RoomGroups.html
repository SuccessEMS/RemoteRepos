Note: To create a room group, users must have access (security permission).

  1. Select  under ‘Room Group’ from the Web Scheduler calendar page.

  1. Name the room group in the Schedule Name field.
  2. Select the desired Building from the dropdown field.

Room numbers will appear in the box on the left of the ‘Create Group’ window.

 

Select either:

  • the desired rooms and Add, or
  •  Add All to select the entire list of rooms.

  Note: The rooms selected appear on the right side of the ‘Create Group’ window once they have been added to the new room group.  

  1. Select Save.

Note: Once the new room group is created and saved, the page will load. Once loaded, the new room group will be visible under the ‘Room Group’ dropdown for the user to select.

Edit Room Group

  1. Select the group to edit from the ‘Room Group’ dropdown box.
  2. Select the  edit tool.         

Edit options:

  • Schedule Name
  • Building
  • Add rooms or Add All
  • Remove rooms or Remove All
  1. Select Save.

  1. Select the schedule group just edited under ‘Room Group’ to view or utilize.

Delete Schedule Group

  1. Select the Schedule Group to delete from the dropdown box under ‘Room Group’ dropdown box.
  2.  Select  delete.

  1. Select Yes in the ‘Please Confirm Deletion’ window to delete.
