Floor Plan Configuration

Indicators Appeared Enlarged for Floorplans of Certain Dimensions

When users opened the Floor Plan Configuration Tool, the indicator icons often appeared too large. 

Fix: Modified functionality so that indicators appear the correct size regardless of browser type or browser window size (i.e., maximized or on half the screen).
