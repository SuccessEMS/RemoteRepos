SSRS Reporting

Create Data Source

Background 

The SSRS reporting engine will require a data source be setup so that a connection can be established to the MeetingPlanner database.  Once this data source is available, all reports can be linked to the one data source.  This way, if the SQL server information ever changes, you will be able to update the connections for all reports by simply updating this data source.

Steps

  1. Browse to your SSRS instance.
  2. Click "New Data Source."

Image_018.png

  1. Create the new data source.
  • Select "Hide in tile view"
  • Update the Connection String to match your SQL server and database names
  1. Click Apply
  2. Update the Security "tab" as necessary (this will typically be more limited than the users that have access to reports)
