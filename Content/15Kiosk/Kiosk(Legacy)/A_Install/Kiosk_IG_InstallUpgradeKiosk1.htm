Installing/Upgrading the EMS Kiosk (Legacy)

Important: Before beginning the installation process, please fulfill installation System Requirements and Prerequisites for EMS Kiosk (Legacy).
  1. Verify that the system requirements and prerequisites have been met.
  2. Load the EMSKiosk.msi file onto the web server that will be running EMS Kiosk (Legacy).
  3. Run EMSKiosk.msi.
  4. The first screen welcomes you to the EMS Kiosk Setup Wizard. 

EMS Kiosk Setup Wizard

  1. Click Next to begin the installation process.  The Destination Folder screen will appear.

Destination Folder

 

 

 

 

 

 

 

 

 

 

 

Note: EMS Kiosk (Legacy) should not be installed in the same physical directory as other EMS web-based products OR under a site running another version of EMS Kiosk.
  1. Enter your EMS Kiosk path and click Next. The Virtual Directory screen will appear.

Virtual Directory

  1. Enter the name you want to give to the virtual directory. Click Next. The Ready to Install screen will appear.

Ready to Install Screen

  1. Click the Install button to install EMS Kiosk (Legacy).
