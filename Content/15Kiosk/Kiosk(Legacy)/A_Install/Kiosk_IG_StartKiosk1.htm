Starting EMS Kiosk (Legacy)

To start the EMS Kiosk (Legacy):

  1. The first time you access EMS Kiosk, you will be redirected to the Config.aspx page (e.g., http://[ServerName]/EMSKiosk/config.aspx). 

Configuration Page

  1. Enter your server and database information. 
  2. Enter your EMS SQL Server name and Database.  
  3. Click Verify Server to continue.
  4. The Kiosk Profile section will appear. 

Kiosk Profile Screen

Note: Kiosk Profiles are configured within the EMS Desktop Client application.  Instructions can be found in the EMS Kiosk Guide.
  1. Select a profile for this Kiosk.
  2. Click Save.
