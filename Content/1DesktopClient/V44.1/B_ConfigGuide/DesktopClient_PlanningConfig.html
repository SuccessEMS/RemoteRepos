Planning Configuration

Process Overview:

To enable end-users to make reservations in EMS, at a minimum, you should configure buildings, rooms, statuses, and groups. We recommend that you configure your system in the following phases:

  • Facilities
  • Resources
  • Video Conferencing
  • Groups
  • Everyday User Applications
  • Billing
  • Administration
  • Settings
    • Parameters
  • Security
  • Tools
  • Import File Specs
  • Other
