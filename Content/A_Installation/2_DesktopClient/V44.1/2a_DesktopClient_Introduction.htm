Introduction

What is the EMS Desktop Client?

The EMS Desktop Client is an application that is installed on administrative users' desktops. Administrative users use this application to configure various workflows, as well as manage day-to-day operations within the application.

Understanding User Types

There are two different user types in EMS.

Administrators: These users create and maintain the EMS application within your organization, and manage or approve usage of space. This user type may also include Service Providers (Catering, Audio Visual, IT Support Staff), Security, and Receptionist Staff. These users will primarily use the EMS Desktop Client.

Everyday Users: This is a majority of the users in the organization. Everyday users are typically requesting rooms and services. These users do not use the Desktop Client, they will primarily use the web based application, VEMS in V44, EMS Web App in V44.1.
