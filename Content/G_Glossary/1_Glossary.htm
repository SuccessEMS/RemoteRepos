Glossary

Academic Import Utility The EMS Academic Import Utility (AIU) is an optional module for EMS Enterprise or EMS Professional that works hand-in-hand with most popular academic scheduling systems to import data on when and where academic classes are scheduled to take place.
Academic Term Defines a set of courses taught for a specific time period, such as a semester. Terms are synchronized into EMS Campus from a Student Information System (SIS).
Academic Unit In EMS Campus an Academic Unit is a level of organization within an institution that defines responsibility for making schedule requests for a portion of a Domain. Academic units are used when assigning permissions to academic users and when filtering reports. In EMS the default term “Academic Unit” can be changed to a more relevant term, such as “Department” or “Division”, if so desired.
