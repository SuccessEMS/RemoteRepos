Requirements

The EMS-RoomWizard Interface web site is typically installed on the server that hosts your VEMS site.  

  • Valid EMS-RoomWizard Interface License
    You must be licensed for the EMS-RoomWizard Interface in order to configure and use this module.  If you are uncertain if your organization is licensed, or if you would like to learn more about it, please contact your Account Executive.
  • RoomWizard Unit(s) Installed/Operational
    RoomWizard unit(s) must be installed and functional.  This can be verified by opening a browser and entering the RoomWizard URL for a specific room (typically http://[RoomWizardIPAddress]/RoomHomePage).
