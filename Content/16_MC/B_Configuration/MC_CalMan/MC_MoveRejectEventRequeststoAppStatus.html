Moving Rejected Event Requests to Approved Status

If you accidentally reject an event request, you can move the rejected request to an approved status.

1. From the Manage Events page select the events that you wish to approve.

Manage Events page

2. Under Actions, click Approve. A message opens asking you if you are sure that you want to approve all selected events.

3. Click OK in the message. A message opens indicating that all selected events were approved.

4. Click OK in the message. The events are approved and are removed from the Rejected Events list. They are now displayed on the Approved Events list.
