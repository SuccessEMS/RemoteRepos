SSRS Configuration Manager

From the SQL server, launch the SQL Server Reporting Services Configuration Manager.  The following items will walk a user through the basic setup for a new reporting server.

Service Account

Network Service will be sufficient.

Web Service URL

This is used as a service URL and can be set to the default options.

Database

The default options can be used; change the accounts and other options as necessary in your organization.

Report Manager URL

This can be configured to any URL that an organization desires.  The reporting URL will need to be input into the MeetingPlanner Web interface after MeetingPlanner is installed.

E-mail Settings

Please enter the required email settings.

Execution Account

Not required.

Encryption Keys

Not required.

Scale-out Deployment

Not required.
