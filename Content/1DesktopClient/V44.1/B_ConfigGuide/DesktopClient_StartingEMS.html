Starting EMS

After has been installed on your desktop, a shortcut icon for the application is placed on your desktop. An option for the application is also available from your Start menu. You can double-click the desktop icon to launch the application, or you can select the option from your Start menu.

The following procedure is written from the perspective of {{product_enterprise}}; however, it is applicable for any {{product}} product ({{product_professional}}, {{product_campus}}, and so on).

1. On your desktop, double-click the icon ,
    or, in the Windows start menu in the Search field, enter and then select the {{product_enterprise}} option.

2. In the User ID field, enter your User ID.

3. In the Password field, enter your password.

Tip: You can change the Server and Database you connect to by right-clicking on the Version number. This launches a pop-up where you can choose a different Server and Database. 

4. Click OK. The {{product_enterprise}} application opens. The main menu is open in the application.
