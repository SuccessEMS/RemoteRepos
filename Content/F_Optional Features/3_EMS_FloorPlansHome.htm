EMS Floor Plans

EMS Floor Plans allows users to search for, view and reserve available space from an interactive floor plan within EMS Web App and EMS Kiosk. To configure and use this component, licensing is required for EMS Desktop Client, EMS Web App and/or EMS Kiosk, and EMS Floor Plans. If you are unsure whether your organization has these licenses, or if you would like to learn more about it, please contact your Account Executive.
