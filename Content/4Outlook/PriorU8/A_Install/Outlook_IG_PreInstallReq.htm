Pre-Installation Requirements

  1. Uninstall any older versions of EMS for Outlook.
  2. The Integration with Exchange Web Service must be installed and operational.  Please see the Integration with Exchange Installation Instructions for information on how to install and configure this component.  You can quickly verify if the service has been installed by opening a browser and entering the following:
  3. http://[ServerName]/ExchangeIntegrationWebService/Service.asmx (replace [ServerName] with the name of your web server)
  4. The Web Service Address will be required when running the EMSForOutlook.msi (outlined in the next section.)
  5. EMS must be configured properly in order to activate the EMS for Outlook for each Outlook® user. 
  6. Verify that the necessary prerequisite software is installed on your users’ workstations.
