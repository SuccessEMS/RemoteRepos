Import Rooms

You can import rooms from a tab-delimited file into a building, area, or view that you select. (See Also: Room Import File Specifications). After you import rooms, you must edit the configuration (setup type, features, images, and so on) for each room.

  1. On the EMS menu bar, click Configuration > Facilities > Rooms. The Rooms window opens.

  1. On the Building dropdown list, select a specific building, a specific area, or a specific view into which to import the rooms.

Tip: If the appropriate building, area, or view is not available, you can configure it.

See Also: Configuring Buildings, Configuring Areas, or Configuring Public Views

  1. Click Import. A dialog box opens, indicating the number of rooms that you are about to import, and asking you if you want to continue.
  2. Click Yes. A message opens indicating that the import was successful.
  3. Click OK to close the message and return to the Rooms window.
  4. For each room that was imported, select the room on the Rooms window, and then click Edit to open the Room in the Room dialog box.
  5. Edit the room configuration as needed. See Also: Configuring Rooms.
