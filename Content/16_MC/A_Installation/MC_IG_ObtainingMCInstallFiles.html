Obtaining Master Calendar Installation Files

Process Overview:

To install EMS Master Calendar, you will:

  1. Obtain and download the installation files (following the steps below).
  2. Run the Master Calendar Installer
  3. Configure Master Calendar to Use the Master Calendar API Web Service
  4. Install/upgrade the Database Component in Master Calendar
  5. Install/upgrade the Master Calendar Web Service.

 

Optional:

  • Verify the Master Calendar API installation.
