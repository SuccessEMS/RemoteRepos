Importing Groups

You can import groups into your EMS database from a tab-delimited text file. (See Import File Specification in the EMS Setup Guide for the file specifications.)

To import groups

  1. On the EMS toolbar, click the Groups icon . The Groups window opens.

  1. Click Import. The Open File dialog box opens.
  2. Browse to and select the text file that lists the groups that you are importing. A dialog box opens, indicating the number of groups that you are about to import, and asking you if you want to continue.
  3. Click Yes. A message opens indicating that the import was successful.
  4. Click OK to close the message and return to the Groups window.
