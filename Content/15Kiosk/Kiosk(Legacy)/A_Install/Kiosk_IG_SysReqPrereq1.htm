System Requirements and Prerequisites for EMS Kiosk (Legacy)

System Requirements  

Supported Browser

Internet Explorer 11, Google Chrome (Latest version)

 

.NET Framework 4.6.1

Prerequisites 

Prior to beginning the installation process, you will need to do the following:

  1. Install or upgrade your EMS databases as outlined in the EMS Installation Instructions.
  2. Manually uninstall any previous versions of the EMS Kiosk on your web server.
