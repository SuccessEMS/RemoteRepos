How Do I Configure Security?

User authentication gates access to your licensed EMS software applications and features. Once you have established and configured authentication, you can configure user security, which controls what users can see and do once they have logged into your EMS applications. Security configuration is done using {{product_dtc}} from a single, convenient menu.
