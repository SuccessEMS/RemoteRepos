Optional EMS Web App Features

If you currently do not own one of optional, separately-licensed modules outlined below, but are interested in more information, please contact your Account Executive.

Integrated Authentication

The Integrated Authentication module is a component for {{product_webapp}} that provides single-sign-on capability using Integrated Windows Authentication, your organization’s portal or LDAP. See Also:Integrated Authentication Configuration Instructions for installation instructions.

Integration with Microsoft® Exchange

Integration with Exchange is a component for {{product_webapp}} that integrates with Exchange. With this module, web users can view the availability of both meeting rooms and attendees, and send Outlook-compatible meeting invitations - all from within {{product_webapp}}. See Also:Integration to Microsoft Exchange for installation instructions.

Floor Plans

EMS Floor Plans allows web users to search for, view and reserve available space from an interactive floor plan within {{product_webapp}}. Floor plans and associated available/unavailable indicator images are configured in the {{product_webapp}} Desktop Client. See Also:Floor Plan Module Installation Instructions.
