API Functions - Kiosk Add Room to Profile and by Setup Type

KioskAddRoomtoProfile

Description

Adds a room to an existing Kiosk Profile.

Description

Type

String

Password *

String

ProfileID*

Integer

RoomID*

Room ID

Integer

Integer

Response Elements

Name

Description

Type

Result

Success or Failure

String

Notes

* Indicates required field.

Possible values for RoomIDType:  0 = Room ID (default), 1 = Room External Reference
