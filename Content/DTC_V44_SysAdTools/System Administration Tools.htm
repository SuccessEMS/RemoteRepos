System Administration Tools

You have several tools at your disposal in EMS to assist you in your administrative duties. These include a tool to update various group attributes such as pricing plans or account name, managing memorized reports, viewing logged-in users and if necessary, log these users off your EMS system, emailing logged-in users, deleting old data in your EMS database, and so on.

This chapter covers the following topics:

  • Change Group Wizard
  • Change Reservation Wizard
  • Room Administration Wizard
  • Managing Memorized Reports
  • Override Existing Bookings Utility
  • Import Reservations Utility
  • Viewing and Emailing Logged-In Users
  • Currency Configuration Wizard
  • Deleting Old Data
  • Changing Booking Dates and/or Statuses
  • Migrating Data

For detailed information the Video Display Interface tool, see Appendix C, “Video Display Interface”.
