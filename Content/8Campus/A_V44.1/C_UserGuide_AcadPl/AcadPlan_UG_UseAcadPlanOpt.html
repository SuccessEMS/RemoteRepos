Using Academic Planning Options in EMS Desktop Client

Academic Planning is an optional component of the EMS software that provides all the functionality that is necessary for Domain Schedulers to ensure that every class meets in a location and at a time that is acceptable to departments and professors. The system provides automatic room assignment within user-supplied parameters, integration with your SIS/ERP software, and efficient online collaboration tools.
