Configuring the EMS Polycom Rooms/Resources

Before activating the EMS Polycom Interface, the EMS video conference room and/or resource list must be synchronized with a Polycom RealPresence Resource Manager (e.g. room or resource). Systems are linked via the EMS Room/Resource External Reference field and the UserID in Polycom.

Rooms can be accessed in EMS within Configuration > Facilities > Rooms. Resources can be accessed in EMS within Configuration > Resources. For information on how to configure an EMS room and/or resource, please refer to the EMS Setup Guide.
