Enable Visitor Management

Visitors, or guest attendees who may not be part of your organization, can be added to your meetings if you enable this feature. Depending on your licensing, you may need to contact your Professional Services representative to add this feature.

To enable visitor management, you will

  • Configure a Category for Visitors, which can be associated with booking templates.
  • Set Everyday User Booking or Process templates to allow the Visitor category.
  • Enable Check-In (optional).
  • Authorize users to manage Visitors.

Configure a Category for Visitors

  1. In the , navigate to Configuration > Administration > Categories.

  1. Select the Display to Everyday Users option on the Everyday User Applications tab so that the category is available for those booking templates.

  1. Configure Everyday User Templates to include the Visitors category you created (on the Categories tab).

  1. Enable Check-In for specific rooms.
  2. Authorize users to manage visitors. If you only allow certain types of users to handle visitors, be sure to assign them to the template you configured above using the Assign button. If you wish to allow visitors to check themselves in (for example, using EMS Mobile App, choose the Self Serve Mode on the Process Template tab when configuring the template, and be sure to enable it for Everyday User Applications as needed.
