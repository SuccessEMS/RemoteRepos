Importing Groups

You can import groups into your EMS database from a tab-delimited text file. (See Also: Group Import File Specifications.)

  1. On the toolbar, click the Groups icon . The Groups window opens.

  1. Click More, and then click Import Groups. The Open File dialog box opens.
  2. Browse to and select the text file that lists the groups that are importing. A dialog box opens, indicating the number of groups that you are about to import, and asking you if you want to continue.
  3. Click Yes. A message opens indicating that the import was successful.
  4. Click OK to close the message and return to the Groups window.
