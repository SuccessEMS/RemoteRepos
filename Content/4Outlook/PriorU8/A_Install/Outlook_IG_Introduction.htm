Introduction

EMS for Outlook is an optional add-in that integrates EMS’ room reservation process directly with Microsoft Outlook 2010/2013.  Users can view room availability in addition to attendee free/busy information simultaneously and book/manage their meetings directly within Outlook.  This document lists the steps you must take to install and configure EMS for Outlook.
