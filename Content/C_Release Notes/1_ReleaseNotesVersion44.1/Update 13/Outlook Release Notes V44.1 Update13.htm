Create Reservation

Users Could Not Add Two+ Services to Multiple Bookings in a Recurring Series

When creating a new recurring reservation, users were unable to add two or more services to multiple bookings. When attempting to add the second service, the Apply to Additional Booking modal was empty. 

Fix: Modified functionality so that users can add two or more services to multiple bookings of recurring series.
