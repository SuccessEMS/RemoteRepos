Configure Video Conference Capabilities

To configure video conferencing capabilities (so that Everyday Users can book video conference meetings in EMS Web App or EMS for Outlook) you will:

  • Configuring Video Conference Room Features
  • Configure Video Conference Services (Categories and Resources)
  • Configure Rooms With Video Conference Equipment.
  • Configure Override Rooms
  • Configure Event Types
  • Configure a Video Conference Booking Template
  • Configure Notifications and Reports

Configuring Video Conference Room Features

A feature is a built-in or permanent attribute of a room, such as a network connection, a built-in plasma TV, window, and so on. Your users can use features as criteria when they are searching for a room. They can also filter the available rooms that can be reserved for an event by room features. After you configure different room features for your organization, you must indicate which rooms have these features.

  1. Configure Room Features in EMS Desktop Client to reflect your video conferencing equipment options, such as a projector or a webcam. For detailed instructions, See Also: Configuring Features.

Tip: Be sure to select Tied to Resource when configuring a video conference Feature. This links the room to video conferencing equipment and services and streamlines the booking process. If you want EMS Web App users to see this feature when they are searching for available space, leave Available to Everyday Users selected.

  1. Associate the feature(s) with Rooms (on the Features tab for Room Configuration); move the feature from the Available panel to the Selected panel as shown below. For detailed instructions, See Also: Configuring Rooms.

Tip: Do not select the Maintain Inventory option when you are configuring this resource.

Configure Video Conference Services (Categories and Resources)

Configure several service categories to represent additional services to be performed for the video conference, such as A/V Setup, and a Mobile Video Conference Cart.

  1. Configure a category for video conferencing services. Click here for detailed instructions on Configuring Categories.
  2. Configure resources and services within the Category for equipment, setup, tear down, etc. For detailed instructions, See Also: Configuring Resources for a Category. Depending on your organization, you may need to configure two types of video conferencing resources for the category that you just created:
  • Built-in video conferencing equipment, such as a Built-in Video Conference Equipment (Note: Do not select the Maintain Inventory option).
  • A Mobile Video Conferencing Cart (Note: Be sure to select the Maintain Inventory option , plus a value in the Quantity Available field).

Tip: You can use the Room Configuration Wizard to mass-assign a feature to multiple rooms in a single step. Additionally, you may need to set the Status for these services to Request if your organization requires approvals.

For detailed instructions, See Also: Room Administration Wizard.

Configure a Video Conference Booking Template

For detailed instructions, see Configuring an Everyday User Booking Template and use the example below.

Tip: Video conferences are typically reserved in EMS Web App due to its streamlined permissions management and ease of use (especially the Calendar view). If you have a significant number of users who book video conferences in EMS Desktop Client, especially if they have limited access and may not be comfortable using the more expert users' booking process, you may wish to create a video conference template for the Reservation Wizard. Familiarize yourself with the process here and then for detailed instructions, See Also: Creating Reservation Wizard templates.

  1. Set the Mode to Self Serve. and the Video Conference option is selected for the template. If your organization requires such meetings to be approved, use Requested Request Status.

  1. Move Categories (Resources and Services) for setup, teardown, equipment, and so forth, as you defined above, into the Selected panel on the Categories tab. Below is an example showing all "VC" (video conference) service categories for an organization enabled for users to select when booking with this template.

  1. Move Rooms (which you defined above) on the Rooms tab to the Selected panel. If your organization requires video conference-equipped rooms to go through an approval process, use the Request panel on the lower right.

Concept: Override Rooms Can "Override" Normal Room Configuration and Setup/Teardown Times

Users will be able to change configuration and start and end time defaults that would normally come from the room definition, in order to accommodate extra video conferencing equipment and services.
  1. Move Event Types (which you defined above) on the Event Types tab to the Selected panel.

  1. On the Video Conference tab, select the video conferencing feature you defined in Step 1 (above).

  2. Complete fields on the remaining tabs as needed and click OK to save.

Configure Notifications and Reports

To set how and when users are notified about the meeting and any changes to it (particularly as equipment and service requests move through the approval process), Configure Notification Rules.

Tips: Notification Rules are highly recommended if you integrate with CISCO TMS.

The most common report run on video conferences is the Setup Worksheet report, set to automatically run daily or weekly, and to deliver to stakeholders' inboxes. For detailed instructions, See Also: EMS Setup Worksheet.
