Web Process Templates

Cannot Add Override Description Room to Video Conference WPT

Users received the error message, "The location you selected is not available for all of your dates," when choosing to override room description in a Web Process Template.

Virtual EMS (VEMS)

Area Description
My Booking Page

Check In Was Not Disappearing from My Booking Page

Once the booking ended in the Mobile Virtual EMS page, the check in icon should have disappeared. 

Fix: Modified functionality so that the check in icon disappears after the booking ends. 

TFS item 30220

EMS API

Area Description
API Call

Booking Cancel Issue

The error, "Invalid URI: The URI is empty," was being generated when the API call, EMSAPI UpdateBooking, was used on a booking made in the EMS for Outlook plugin.

Fix: Modified functionality so that no error is received.

TFS item 25389

July 14, 2017 201705170 43.0.165.165

Virtual EMS (VEMS)

Area Description
Create Reservation

Building Hours Violation Not Working for Unauthenticated Users 

Unauthenticated users who created a reservation with a start time that violated the building's hours of operation did not receive a violation of building hours error message. 

Fix: Modified functionality so that unauthenticated users choosing a start time that violates the building's hours of operation receive an error message. 

TFS item 29832

43.0.165.165

Virtual EMS (VEMS)

Area Description
Create Reservation

When a Required Field Was Left Blank, Error Read As "Undefined"

While creating a reservation, users who left a required field blank (e.g., Event Type) were receiving an "Undefined" error message instead of alerting the user that the field needed to be completed. 

Fix: Modified functionality so that an error message includes the correct verbiage to alert the user that the field needs to be completed. 

TFS item 27583
