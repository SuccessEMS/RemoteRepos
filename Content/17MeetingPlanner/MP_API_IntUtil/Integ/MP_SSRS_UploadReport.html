Background

Emergingsoft will be posting various reports that we think may be helpful to our customers, and will provide assistance updating or customizing reports.  If you choose to download a report or receive a .rdl file from our support team, please see the instructions below for how to upload the report to your SSRS instance.

Steps

1)  Browse to your SSRS instance.

2)  Click "Upload File."

3)  Upload the report (.rdl).

4)  From the dropdown menu for the report, click "Manage."

5)  Click "Data Sources" > "A shared data source"

6)  Browse to your current MeetingPlanner data source.

*In the event that you cannot browse to your shared data source, you can manually type in the path.  The default path would be "/MeetingPlanner" (no quotes).

7)  Click "Apply."

The script only needs to be run against the database once, so if you are uploading a report and have already run the script, there is no need to re-run it.  (Multiple executions will not have an impact on any data).
